## Image Specifications

### Profile Picture (`profile.jpg`)
- **Dimensions**: 300x300px minimum (square aspect ratio)
- **Format**: JPG or PNG
- **Size**: Under 500KB for optimal loading
- **Content**: Professional headshot or avatar

### Hero Background (`hero.jpg`)
- **Dimensions**: 1920x1080px minimum
- **Format**: JPG
- **Size**: Under 1MB for optimal loading
- **Content**: Abstract, tech-related, or professional background

### Project Images (`project1.jpg`, `project2.jpg`, `project3.jpg`)
- **Dimensions**: 600x400px minimum (3:2 aspect ratio)
- **Format**: JPG or PNG
- **Size**: Under 300KB each for optimal loading
- **Content**: Screenshots, mockups, or representative images of the projects

## Image Optimization Tips

1. **Compress images** before uploading to reduce file sizes
2. **Use appropriate formats**: JPG for photos, PNG for graphics with transparency
3. **Consider WebP format** for modern browsers (optional)
4. **Add alt text** in HTML for accessibility
5. **Use lazy loading** for better performance

## Adding New Images

1. Place image files in this `assets/` directory
2. Update the `src` attributes in `index.html` if needed
3. Ensure file names match those referenced in the HTML
4. Test that images load correctly on all devices
